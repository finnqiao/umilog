# Manual Screenshot Steps (Remaining Flows)

Use the Simulator with `UmiLog` running. Save captures into `docs/screens/`.

1) Map Search
- Tap magnifier icon on map.
- Capture as `docs/screens/13-map-search.png`.

2) Map Filters
- Tap filter sliders icon in map surface.
- Capture as `docs/screens/14-map-filters.png`.

3) Site Inspect
- Tap a site row in the map list to open Inspect.
- Capture as `docs/screens/15-site-inspect.png` (medium) and `docs/screens/16-site-inspect-expanded.png` (expanded by dragging sheet up).

4) Plan Sheet
- From Inspect, tap "Plan".
- Capture as `docs/screens/17-plan.png`.

5) Featured Destination
- Delete app + relaunch to trigger first-run featured card (if enabled).
- Capture as `docs/screens/18-featured-destination.png`.
